Тест

1) Комплекс программных и аппаратных средств для обеспечения
автоматизации производства и других сфер жизнедеятельности человека,
включающий в качестве составной части кабельное и сетевое оборудование
называется:

  1. Распределенной базой данных.
  2. Информационно-вычислительной сетью.
  3. Телекоммуникационной системой.
  4. Сетевой системой управления базами данных.

2) Процесс установления подлинности пользователя информационно-
вычислительной сети называется:

  1. Регистрацией.
  2. Авторизацией.
  3. Аутентификацией.
  4. Контролем доступа ресурсов.

3) Степень свободы действия пользователя (просмотр, использование, владение)
по отношению к данному ресурсу называется:

  1. Правами доступа к ресурсу.
  2. Администрированием ресурса.
  3. Свойствами учетной записи пользователя.
  4. Аудитом доступа к ресурсу.

4) Возможность наращивания мощности вычислительного устройства
(количество и быстродействие процессоров, объем оперативной и внешней
памяти) для пропорционального увеличения скорости и плотности
(определенное количество запросов в единицу времени) обработки запросов, а
также объемов хранимой информации называется:

  1. Отказоустойчивостью.
  2. Дублированием основных аппаратных компонентов.
  3. Масштабированием.
  4. Технологией RAID.

5) Изолированная медная жила, помещенная в медную оплетку, покрытая
гибкой изоляционной оболочкой представляет собой:

  1. Кабель на основе «витых пар»
  2. Одномодовый оптоволоконный кабель.
  3. Коаксиальный кабель.
  4. Многомодовый оптоволоконный кабель.

6) Укажите устройство относящееся к коммутационному оборудованию сети:

  1. Модем
  2. Сетевая плата (Ethernet).
  3. Концентратор.
  4. Сетевая плата (Token-Ring).

7) Оборудование, служащее для обеспечения согласования разных технологий
передачи данных, создания переключаемых каналов между узлами сети и
организации виртуальных ЛВС называется:

  1. Концентратором
  2. Маршрутизатором.
  3. Коммутатором.
  4. Модемом xDSL

8) Программное обеспечение, предназначенное для выполнения обслуживания
информационно-вычислительной сети (прежде всего ее аппаратного
обеспечения), относится к:

  1. Дополнительному ПО.
  2. Прикладному ПО.
  3. Средствам сетевого управления.
  4. Системному ПО

9) Уровень программного обеспечения, отвечающий за разграничение прав
доступа пользователей и проверку полномочий при доступе к
информационным ресурсам через уровень представления, называется:

  1. Уровнем бизнес-правил.
  2. Уровнем хранения и извлечения информации.
  3. Уровнем именования и идентификации.
  4. Уровнем безопасности.

10) Модель вычислений, реализуемая в программном обеспечении, при которой
обработка и хранение данных осуществляется на двух и более
вычислительных устройствах, называется:

  1. Локализованной моделью вычислений
  2. Распределенная модель вычислений.
  3. Модель вычислений на основе файлового хранилища.
